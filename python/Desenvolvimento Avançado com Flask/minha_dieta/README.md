# Minha Dieta API

> Neste desafio, você vai desenvolver uma API completa para controle de dieta diária usando Flask. A aplicação permitirá aos usuários registrar suas refeições com informações detalhadas como nome, descrição, data/hora e se está dentro da dieta.
> O sistema oferecerá funcionalidades completas de CRUD (criar, ler, atualizar, deletar) para gerenciamento das refeições, com persistência em banco de dados.
> Este projeto reforçará de forma prática os conceitos avançados de Flask, proporcionando uma experiência real de desenvolvimento de APIs RESTful com estruturação autônoma de rotas e propriedades.
